## S.2 Functionality
- **[S.2.1] Continuous Monitoring:** The system continuously monitors the home environment for defined events.  
- **[S.2.2] Automatic Dispatch:** The system selects the specific emergency service based on the event type (e.g., fire department for fire events).  
- **[S.2.3] Visual Verification Integration:** The system accesses security camera feeds to verify alerts.
