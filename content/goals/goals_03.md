## G.3 Expected benefits
The deployment of this system is expected to provide the following benefits:  
- **[G.3.1] Automated Emergency Response:** Automatic calling of police, fire, or ambulance services based on the specific type of event detected.  
- **[G.3.2] Continuous Protection:** The system guarantees 100% uptime for constant protection.  
- **[G.3.3] Visual Verification:** Integration with security cameras allows for visual verification of events.  
- **[G.3.4] Personal Safety:** Specific support for medical emergencies (e.g., falls) via wearable controllers.
