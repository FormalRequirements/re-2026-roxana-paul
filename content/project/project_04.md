## P.4 Tasks and deliverables

[P.4.1] The project tasks are focused on achieving the submission deadline with the following key outcomes:  

- **Tasks:**  
    - Develop and validate the project.  
    - Perform system verification.

- **Deliverables:**  
    - **Release v1:** The first executable version of the system.  
    - **Final Project:** Complete project documentation and source code.
